# 🧭 Application Architecture Guidelines

Guía general para estructurar aplicaciones fullstack de forma mantenible, escalable y clara.

Este repositorio reúne lineamientos para frontend, backend y comunicación entre capas.

## 📖 Cómo usar esta guía

- Puede aplicarse total o parcialmente según el proyecto
- No es necesario implementar todo desde el inicio
- Se recomienda comenzar simple y evolucionar conforme crece la aplicación
